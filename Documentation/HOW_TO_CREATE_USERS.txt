Q: What's the default password for tinker-created users?
A: "password" (without quotes)

Q: Can I change the password?
A: Yes, use the profile page after login or:
   php artisan tinker
   >>> $user = \App\Models\User::find(1);
   >>> $user->password = bcrypt('newpassword');
   >>> $user->save();

Q: How do I promote a user to admin?
A: Method 1: Admin can edit user and change role
   Method 2: php artisan tinker
            >>> $user = \App\Models\User::where('email', 'user@test.com')->first();
            >>> $user->role = 'admin';
            >>> $user->save();

Q: Can I delete all users and start over?
A: Yes! Run: php artisan db:wipe --force
   Then: php artisan migrate --force

Q: What if I forget a user's email?
A: php artisan tinker
   >>> \App\Models\User::all(['id', 'name', 'email', 'role'])
